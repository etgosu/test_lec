# 주식 종목 리서치 웹 앱 — Spec

## 개요
좌측 사이드바에서 종목을 추가·삭제·선택하면 우측 본문에 해당 종목의
현재가·등락율·일중 범위·차트·뉴스 헤드라인이 표시되는 개인용 종목 뷰어.
한국(.KS) 및 미국 주식을 동일한 인터페이스로 지원한다.

## 범위

### 포함
- 사이드바: 종목 목록 표시, 종목 추가(코드 입력), 종목 삭제(×)
- 본문: 선택된 종목의 현재가 / 등락율 / 등락 방향 색상 구분
- 본문: 금일 최저가 / 최고가
- 본문: 인트라데이 차트 (당일 가격 흐름)
- 본문: 뉴스 헤드라인 목록 (제목 + 출처, 최대 5개, 클릭 시 원문 링크)
- 종목 목록: 세션 상태로 관리 (새로고침하면 초기화)

### 제외
- localStorage / 서버 영속성 — 세션이 끝나면 목록 초기화
- 재무지표(PER·PBR 등) — 1차 검증 후 추가
- 차트 기간 선택 (주·월·연) — MVP는 당일만
- 알림·포트폴리오 추적 — 별개 도구 영역
- AI 요약 — 핵심 뷰 증명 후 선택적 추가

## 시나리오

### 1. 미국 주식 추가 후 상세 조회
- **Given** — 사이드바가 비어있는 초기 상태
- **When** — `AAPL` 입력 후 추가 → 사이드바의 AAPL 항목 클릭
- **Then** — 본문에 현재가, 등락율(색상 포함), 금일 최저/최고가,
  당일 가격 선 차트, 뉴스 헤드라인 최대 5개가 표시된다

성공 기준:
- [ ] `AAPL` 추가 → 사이드바에 "AAPL" 또는 "Apple Inc." 텍스트가 존재한다
- [ ] AAPL 항목 클릭 → 본문에 현재가가 숫자로 표시된다 (USD 단위)
- [ ] 등락율이 `+1.23%` / `-1.23%` 형식으로 표시된다
- [ ] 상승이면 초록, 하락이면 빨강으로 등락 색상이 구분된다
- [ ] 금일 최저가와 최고가가 함께 표시된다
- [ ] 당일 시가 이후 현재까지의 가격 선 차트가 표시된다
- [ ] 뉴스 헤드라인이 1~5개 표시되고, 각 항목에 출처명이 있다
- [ ] 뉴스 헤드라인 클릭 시 원문 URL로 이동한다

### 2. 한국 주식 추가 — 6자리 숫자 자동 감지
- **Given** — 사이드바 초기 상태
- **When** — `005930` 입력 후 추가 (suffix 없음) → 항목 클릭
- **Then** — `.KS`가 자동으로 붙어 삼성전자 데이터가 조회되고,
  시나리오 1과 동일한 정보가 본문에 표시된다 (단위: KRW)

성공 기준:
- [ ] `005930` 추가 → 사이드바에 "삼성전자" 또는 `005930.KS` 관련 텍스트가 존재한다
- [ ] 항목 클릭 → 본문 가격이 KRW 단위로 표시된다
- [ ] 시나리오 1의 현재가·등락율·차트·뉴스 기준이 동일하게 충족된다

### 3. 존재하지 않는 종목 추가 시도
- **Given** — 사이드바 상태
- **When** — `XXXXXXX` 등 유효하지 않은 코드 입력 후 추가
- **Then** — 에러 메시지가 표시되고 사이드바 목록에는 추가되지 않는다

성공 기준:
- [ ] 유효하지 않은 코드 추가 시도 → 에러 메시지가 표시된다
- [ ] 사이드바 종목 목록에 해당 코드가 추가되지 않는다

### 4. 종목 클릭 후 로딩
- **Given** — 사이드바에 종목이 1개 이상 있는 상태
- **When** — 종목 항목을 클릭하고 데이터를 기다리는 동안
- **Then** — 본문에 로딩 인디케이터(스켈레톤)가 표시된다

성공 기준:
- [ ] 항목 클릭 직후 본문에 로딩 상태가 표시된다
- [ ] 데이터 수신 완료 후 로딩이 사라지고 결과가 나타난다

### 5. 종목 추가 — 사이드바 목록 반영
- **Given** — 사이드바에 AAPL이 없는 상태
- **When** — `AAPL` 입력 후 추가 확인
- **Then** — 사이드바 목록에 AAPL 항목이 즉시 추가된다

성공 기준:
- [ ] 유효한 코드 추가 → 사이드바에 해당 종목 항목이 나타난다
- [ ] 추가된 항목에 종목 코드 또는 종목명이 표시된다
- [ ] 동일 코드를 중복 추가해도 목록에 하나만 존재한다

### 6. 종목 삭제
- **Given** — 사이드바에 종목이 1개 이상 있는 상태
- **When** — 특정 종목 항목의 × 버튼 클릭
- **Then** — 해당 종목이 사이드바에서 사라진다.
  삭제한 종목이 현재 본문에 표시 중이면 본문이 비워진다

성공 기준:
- [ ] × 클릭 → 해당 종목 항목이 사이드바에서 사라진다
- [ ] 현재 선택된 종목을 삭제하면 본문의 상세 정보가 사라진다
- [ ] 선택되지 않은 종목을 삭제하면 본문은 유지된다

## 불변 규칙

**등락 색상**: 모든 시나리오에서 상승은 초록(green), 하락은 빨강(red)으로 표시된다.
**뉴스 상한**: 헤드라인은 최대 5개만 표시된다 (Yahoo Finance 제공량에 무관).
**한국 종목 자동 감지**: 숫자 6자리 입력 시 `.KS`를 자동으로 붙여 조회한다.
이미 suffix(`.KS`, `.KQ` 등)가 있으면 그대로 사용한다.
**세션 휘발성**: 종목 목록은 세션 상태로만 관리된다. 새로고침하면 초기화된다.
