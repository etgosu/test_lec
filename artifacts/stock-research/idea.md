# 주식 종목 리서치 도구

## Problem Statement
HMW — 종목을 공부할 때, 여러 탭을 오가지 않고 한 화면에서
가격 맥락과 뉴스를 한 번에 파악할 수 있을까?

## Recommended Direction
종목 코드를 입력하면 시장 데이터(현재가·등락율·차트·일중 범위)와
뉴스 헤드라인이 한 화면에 모이는 개인용 종목 뷰어.

`yahoo-finance2` (npm, 인증 불필요)로 한국(005930.KS)·미국(AAPL) 주식을
동일한 API로 수집한다. 인증·API 키 없이 바로 동작하므로 로컬에서
즉시 실행 가능하다.

## Key Assumptions to Validate
- [ ] yahoo-finance2로 KS·US 종목의 현재가·등락율·일중 고저·차트·뉴스를
      안정적으로 가져올 수 있다 → 005930.KS + AAPL 샘플 pull로 즉시 확인
- [ ] 뉴스 헤드라인 정도면 공부 진입에 충분하다 (본문 전달 불필요)
- [ ] Yahoo Finance rate limit이 개인 단일 사용자 사용에 문제없다

## MVP Scope
포함:
- 종목 코드 검색 (KS 자동 감지 또는 수동 suffix 입력)
- 현재가 + 등락율 + 등락 방향 색상 구분
- 금일 최저가 / 최고가
- 인트라데이 차트 (당일 가격 흐름)
- 뉴스 헤드라인 목록 (제목 + 출처, 클릭 시 원문 링크)

데이터 소스: `yahoo-finance2` — 인증 없음, 한국·미국 주식 모두 지원

제외:
- 재무제표·밸류에이션 지표 — 1차 검증 후 추가
- AI 요약 — 핵심 뷰 완성 후 선택적 추가
- 알림·포트폴리오 추적
- 다사용자·공유 기능

## Not Doing (and Why)
- AI 요약 — 데이터 뷰 유용성 먼저 검증; 복잡도↑ API 비용 발생
- 재무지표 (PER 등) — 가격 맥락이 1차 가설; 밸류에이션은 이후
- 실시간 호가창 — 15분 지연으로도 공부 목적엔 충분
- 차트 기간 선택 — MVP는 당일 흐름만; 주·월은 이후
